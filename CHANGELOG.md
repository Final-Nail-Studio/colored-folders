# Changelog

## [1.1.0] - 2026-03-08

### Added
- Configurable horizontal alpha gradient (left/right alpha sliders in settings)
- Selected folder alpha multiplier setting
- Optional colored folder label text
- Tinted folder icons at configurable opacity

### Fixed
- Blurry text rendering when Color Folder Labels is enabled
- Duplicate white folder icon appearing in flat list view
- Icon and text alignment in both tree and flat list views

## [1.0.0] - 2024-01-01

### Added
- Folder background tinting in Project window based on folder name
- Case-insensitive folder name matching
- Recursive color inheritance with nearest-ancestor resolution
- Alt + Right Click context menu with 16-color palette
- Recursive toggle per rule
- Settings window (Tools > Colored Folders > Settings)
- Preset import/export as JSON
- Dark and light theme support
- Persistent project-local settings in ProjectSettings/ColoredFoldersSettings.json
- Sample preset files
