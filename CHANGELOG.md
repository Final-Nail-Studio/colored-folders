# Changelog

## [1.0.0] - 2024-01-01

### Added
- Folder background tinting in Project window based on folder name
- Case-insensitive folder name matching
- Recursive color inheritance with nearest-ancestor resolution
- Alt + Right Click context menu with 16-color palette
- Recursive toggle per rule
- Settings window (Tools > Colored Folders > Settings)
- Preset import/export as JSON
- Dark and light theme support
- Persistent project-local settings in ProjectSettings/ColoredFoldersSettings.json
- Sample preset files
